# Звіт з практики: Розробка кастомних секцій Shopify

Цей репозиторій містить результати виконання практичних завдань зі створення та оптимізації кастомних функціональних елементів для теми Shopify з використанням Liquid, CSS та JavaScript.

## Виконані практичні завдання

Логіка завдань розбита на окремі модульні файли:

* **`sections/custom-cart-drawer.liquid`** — Кастомний боковий кошик.
* **`sections/custom-quick-view.liquid`** — Модальне вікно швидкого перегляду.
* **`sections/main-collection-ajax.liquid`** — AJAX-колекція з фільтрами.
* **`sections/custom-swatches.liquid`** — Кастомний вибір кольору.
* **`sections/custom-sticky-cart.liquid`** — Липка кнопка "Додати в кошик".
* **`sections/custom-size-guide.liquid`** — Таблиця розмірів.
* **`sections/custom-banner.liquid`** — Банер з приховуванням.
* **`sections/email-signup-banner.liquid`** — Банер для Email-підписки.
